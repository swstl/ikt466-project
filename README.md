# Getting Started

## Installation

Install [uv](https://github.com/astral-sh/uv):

## Create venv and sync Dependencies

Sync changes from `pyproject.toml` to get the newest environment:

```sh
uv sync
```

## Add New Packages

Add new packages with:

```sh
uv add <packagename>
```

## run the initialisation:
uv run python init.py
