# Getting Started

## Installation

Install [uv](https://github.com/astral-sh/uv):

## Create a Virtual Environment

```sh
uv venv
```

## Sync Dependencies

Sync changes from `pyproject.toml` to get the newest environment:

```sh
uv sync
```

## Add New Packages

Add new packages with:

```sh
uv add <packagename>
```
